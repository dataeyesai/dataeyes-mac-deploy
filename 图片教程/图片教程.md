# DataEyes × OpenClaw 安装图片教程

## 步骤 1：确认安装包内容

![步骤 1](01.png)

先确认桌面上的安装包内容完整。

## 步骤 2：等待 OpenClaw 安装完成

![步骤 5](05.png)

安装器会自动检查 Node.js，并安装 OpenClaw。

当前版本的安装策略是：
- 优先直接执行 `npm install -g openclaw@latest`
- 如果当前环境没有全局安装权限，会自动回退到 `~/.npm-global`

## 步骤 3：输入 DataEyes API Key

![步骤 6](06.png)

按照提示输入你自己的 DataEyes API Key。

## 步骤 4：等待自动写入配置并启动 Gateway

![步骤 7](07.png)

安装器会自动：
- 写入 `~/.openclaw/openclaw.json`
- 安装或启动 Gateway
- 做一次验证

## 步骤 5：安装完成后自动打开本机控制台

![步骤 8](08.png)

默认控制台地址：
- `http://127.0.0.1:18789`

## 补充说明

- 默认模型：`dataeyes/gpt-5.4`
- 自动回退：`dataeyes/gemini-3.1-pro-preview-customtools`
- 自动回退：`dataeyes/claude-opus-4-6`
- 如果当前终端找不到 `openclaw`，先执行：

```bash
export PATH="$HOME/.npm-global/bin:$PATH"
```
